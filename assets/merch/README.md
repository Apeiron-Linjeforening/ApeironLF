# Merch-bilder

Plasser bilder for merch-produkter her. Navn og forslag til bilder:

| Filnavn | Produkt | Foreslått beskrivelse |
|---------|---------|----------------------|
| `segl-pin.svg` | Segl-pin | Emaljert pin i segl-design (bestseller) |
| `gresk-pin.svg` | Gresk tekst-pin | Pin med ΑΠΕΙΡΟΝ-tekst |
| `genser.svg` | Crew-neck genser | Marineblå genser med broderi |
| `tote-bag.svg` | Tote-bag | Beige canvas bag |
| `filosofkrus.svg` | Filosofkrus | Hvit keramikk krus |
| `båt.jpg` | Båt-pin | Båt-design pin (jpg-format) |

## Format
- Filformat: `.svg`, `.jpg`, `.png` eller `.webp`
- Anbefalt størrelse: **800×800px** (kvadratisk)
- **Autocrop**: `object-fit: cover` sentrerer bildet (crop-center)

## Justering (om trengs)
- For å justere "sentrering": endre `object-position` i CSS (f.eks. `object-position: center top;`)

## Endre produkter
Rediger `merch-data.js` i prosjekt-roten. Se kommentarer i denne filen.
